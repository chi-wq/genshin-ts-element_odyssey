元素试炼:深渊回廊 - 玩法介绍

关卡GUID:24439692738

简介
本游戏是一款闯关爬塔式元素战斗挑战图，在奇域中操控角色收集元素球、触发反应、击倒敌人逐关推进。
活用元素反应击倒敌人。击杀敌人后场上元素球会变为可拾取状态，触碰可更新主/副元素，使得普攻可触发反应。攻击分两段：副元素附着 → 主元素一击，组合不同元素即可触发6种元素反应，反应击杀可获得高分。未拾取的元素球会变为深渊球，触碰会触发对我方的追踪攻击。全灭敌人并集齐指定数量的元素球即可通关，倒计时归零则失败。

详情

基本规则

1. 击杀敌人 → 场上元素球发亮，进入可拾取状态（5秒）
2. 触碰元素球 → 主/副元素切换，+30分
3. 攻击 → 副元素附着 → 主元素一击，触发元素反应（6种），反应击杀+100分
4. 集齐元素球 + 全灭敌人 → 通关

元素球超时变为深渊球，触碰会触发敌方追踪攻击。可用净化道具或击杀敌人恢复为可拾取。
特殊球（风环/岩盾/草色十字/光柱）不计入通关球数，触碰即生效：加时/护盾/回血/全灭。

计分

普通击杀 +1
元素反应击杀 +100
收集元素球 +30

分数跨关卡累积，最终结算时显示。
